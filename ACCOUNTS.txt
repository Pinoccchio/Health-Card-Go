========================================
ROLE STRUCTURE
========================================

1. SUPER ADMIN (role: super_admin)
   - Full system access
   - Create all user types
   - Manage all services and records

2. HEALTHCARE ADMIN (role: healthcare_admin)
   - Service-specific access via assigned_service_id
   - Manage appointments for assigned service only
   - Create/edit/delete announcements
   - Access service-specific records

3. STAFF (role: staff)
   - Disease surveillance for ALL diseases
   - Enter disease data (Dengue, Measles, Rabies, Malaria, etc.)
   - Generate disease reports
   - NO appointment management access

4. DOCTOR (role: doctor) - DEPRECATED
   - Kept for backward compatibility
   - View ALL records during appointments
   - Create medical records

5. PATIENT (role: patient)
   - Book appointments (max 2 services)
   - View own records
   - Download health card
   - Submit feedback
